# MVC E-Commerce System

## Technologies Used

- **ASP.NET MVC 5.2.9** - Web application framework
- **.NET Framework 4.7.2** - Runtime framework
- **C#** - Programming language
- **Bootstrap 5.2.3** - Frontend CSS framework
- **jQuery 3.7.0** - JavaScript library
- **jQuery Validation** - Form validation
- **Newtonsoft.Json** - JSON serialization
- **Razor Views** - View engine
- **IIS Express** - Web server

## Description

An e-commerce web application built with ASP.NET MVC that provides a complete online shopping experience. The system features user registration and authentication, a product catalog, shopping cart functionality, and order management. It includes separate interfaces for customers and administrators, with the admin panel allowing management of products, orders, customers, and reports. The application uses in-memory caching for data storage and loads product data from JSON files.
